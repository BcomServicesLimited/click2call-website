Setup Process (Step by Step) Recommended Setup: Small Business with Reception Call Flow For businesses wanting a main number that rings reception first, then overflows to other staff, we recommend: 1 × Inbound Business Number ($10/mo ex GST) as the primary inbound line 1 × Cloud PBX User per staff member who needs to make/receive calls ($25/mo ex GST each) This setup allows group mailboxes, after-hours routing, and ring groups to be configured on the main line independently of individual user extensions. It provides greater flexibility for call flow programming and future scalability – staff can be added or removed from the call flow without changing the main number's configuration. Step-by-Step Setup Process From the moment a customer signs up to the moment their first call comes in, here is exactly what happens: Create Your Account: Complete the signup form at portal.click2call.com.au/join. The form takes around five minutes. The account will be activated within 24 hours. Add a Temporary Number: Once active, add a temporary local number from the portal so you can start building the system immediately. Set Up Your Users: Add each member of the team as a user in the portal. Configure call routing, ring groups, or auto-attendant menus. Install the Apps: Download and install the Click2Call softphone on each team member's PC or configure physical handsets. Test the System: Make test calls to and from the temporary number to ensure all routing and apps are working correctly. Divert Your Old Number: Set up a call diversion on the old phone system, pointing the main business number to the new temporary Click2Call number. The customer is now live on the new system. Port Your Number: Submit a port request to move the main number to Click2Call permanently. This takes 5–10 business days. Because of the diversion, there is zero downtime. Remove Temporary Number: Once the port completes, the main number is officially on Click2Call. The customer can cancel their old service and remove the temporary number from their account. 7. Microsoft Teams Calling (Direct Routing) 12.1 Product Overview and Architecture Click2Call Microsoft Teams Calling enables businesses to make and receive external phone calls directly within the Microsoft Teams application. The service uses Direct Routing architecture, which is a Microsoft-approved method for connecting Teams to the public switched telephone network (PSTN). Unlike basic SIP trunk providers that only supply a phone number and a connection, Click2Call integrates a full Cloud PBX beneath the Teams interface. This architectural difference means customers receive enterprise-grade PBX features natively, without needing third-party add-ons or complex integrations. The core value proposition is providing two systems for one price: customers get the familiar Microsoft Teams interface powered by Click2Call's advanced Cloud PBX infrastructure. 12.2 Pricing and Commercial Terms Click2Call operates on a transparent, prepaid credit model with no hidden fees.
