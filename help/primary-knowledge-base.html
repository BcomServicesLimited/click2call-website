Primary Knowledge Base - Source of Truth

This document contains the authoritative pricing, calling rates, porting fees, and setup costs for Click2Call services. It is the primary source of truth for all customer inquiries.

1. Monthly Plan Pricing

Cloud PBX User Seat: $25.00 + GST per user per month.

  • Includes a local landline number.
  • Full Cloud PBX features.
  • Unlimited inbound calls.
  • 300 outbound minutes (to NZ and AUS) included per month. After 300 minutes, calls are charged at normal outbound rates.

2. Calling Rates

Outbound Call Rates

  • 1300 - Outbound: $0.50 flagfall, $0.00 per minute.
  • Mobile (NZ and AUS): $0.00 flagfall, $0.20 per minute.
  • Landline (NZ and AUS): $0.00 flagfall, $0.10 per minute.

Inbound Call Rates

  • 1300 Inbound: $0.00 flagfall, $0.20 per minute.
  • 1800 Inbound: $0.00 flagfall, $0.20 per minute.
  • Landline: $0.00 flagfall, $0.00 per minute.

3. Number Porting Fees

  • Local Number Port: $50.00 + GST.
  • 1300 or 1800 Number Port: $100.00 + GST.

Porting typically takes 5-10 days. A copy of the phone bill showing the number to be ported is required. The number will be pre-provisioned on the account on the day of the port to allow time to set up the call flow.

4. Changeover / Setup Fees

  • Self-Setup: Customers can set up their own account for free using our Help Centre articles or by emailing support.
  • Professional Setup: Available for customers who want us to do the setup for them (charges apply, see professional setup charges).

5. Add-on Costs

  • Extra Business Number (Cloud PBX Line): $10.00/mo per line (ex GST). Includes shared numbers, reception, or extra inbound routing, unlimited inbound calling, and full Cloud PBX features.
  • Direct Dial Number (DDI): $5.00/mo (ex GST). No Cloud PBX features.
  • 1300 Numbers: $10.00/mo (inbound only). Requires a Cloud PBX extension for outbound calling.
  • MS Teams Line: $25.00 /user/month (ex GST). Paid to Click2Call. Includes your Australian number, 300 outbound minutes, and unlimited inbound calls.
  • SIP Trunk: $50.00/mo per trunk (ex GST). Includes 500 mins outbound, unlimited inbound calls, unlimited concurrent channels, and is compatible with all major PBXs.
  • Pre-configured VoIP Phones: Yealink T43U = $450.00 + GST, Yealink W76P = $400.00 + GST. Price includes configuration and postage.

6. Free Trial Details

30-Day Free Trial: No credit card required. Customers can create an account, view the portal, and add internal extensions to try out calling features.

7. Other Key Facts

  • All pricing on contracts is exclusive of GST.
  • Mobile numbers are not available for purchase.
  • Channel pricing is no longer needed as each Cloud PBX user has inbuilt channels.
