Add a Temporary Number: Once active, add a temporary local number from the portal so you can start building the system immediately. Test the System: Make test calls to and from the temporary number to ensure all routing and apps are working correctly. Divert Your Old Number: Set up a call diversion on the old phone system, pointing the main business number to the new temporary Click2Call number. The customer is now live on the new system. Port Your Number: Submit a port request to move the main number to Click2Call permanently. This takes 5–10 business days. Because of the diversion, there is zero downtime. Microsoft Teams Direct Routing setup - Outbound calling via Teams (unless customer purchases Teams User Plan) - Integration assistance or configuration support 13.3 Microsoft Teams Requirements To use Direct Routing, customers must have: -
